Manage Sections
Add or remove website sections. They appear instantly in all navigation menus and article / video dropdowns. Removing a section does not delete its articles.
Add New Section
Section names are capitalised automatically. Adding a section makes it available immediately in article & video dropdowns. Removing a section only removes the navigation link — existing articles stay in Firestore.
⚠ Required: Firestore Rules
If sections show 0 or you get errors, your Firestore rules need to allow access to the sections collection. In the Firebase Console go to Firestore → Rules and make sure you have:
match /sections/{doc} {
  allow read, write: if request.auth != null;
}
